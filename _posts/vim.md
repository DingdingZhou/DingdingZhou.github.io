## 基本
h j k l : 这四个键前面都可以加数字,表示移动的次数

## 行
* ^ : 行首
* $ : 行尾
* G : 末行
* nG : n行
* gg : 首行

## 窗口
* H : 当前窗口的首行
* M : 当前窗口的中间
* L : 当前窗口的尾行
* ctrl-b : 等效于pgup,b-backword
* ctrl-f : 等效于pgdn,f-forward
* ctrl-u : 上翻半页
* ctrl-d : 下翻半页 

## 单词/句/段落
* w : 下一个单词词首
* e : 下一个单词词尾
* b : 前一个单词的词首
* ge : 前一个单词的词尾

* ) : 下一句
* ( : 上一句
* } : 下一段
* } : 上一段

# 跳转
* ctrl-o : 视图前进
* ctrl-i : 视图后退
